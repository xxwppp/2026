# 青龙面板环境变量管理插件

管理青龙面板的环境变量，支持多项目、多账号、多实例。

## 配置

在 AstrBot WebUI → **插件管理** → 点击插件卡片上的**配置**按钮 → 点击 **添加** 添加青龙实例。

每个实例包含：

| 字段 | 说明 |
|------|------|
| `name` | 实例名称（如 青龙1） |
| `ql_url` | 青龙面板地址（如 `http://192.168.1.100:5700`） |
| `client_id` | 青龙面板应用 ID |
| `client_secret` | 青龙面板应用密钥 |

支持添加多个实例，配置保存后重启 AstrBot 生效。

## 命令列表

### `/项目`（主菜单）
所有功能的入口，显示菜单：

```
1.项目列表
2.查询
3.管理项目
4.新增项目（管理员使用）
5.删除项目（管理员使用）
6.项目帮助
输入序号（输入q退出）
```

选择序号后进入对应功能，操作完成后自动返回主菜单。

---

### `/项目列表`
直接查看所有项目（按青龙实例分组），非交互式。

### `/新增项目`
创建一个新的环境变量项目。

交互流程：
1. 输入**项目名称**（如 `jd_scripts`，`q` 退出）
2. 选择**青龙实例**（仅多实例时出现，输入序号，`q` 退出）
3. 输入**青龙环境变量名**（如 `JD_COOKIE`，`q` 退出）
4. 输入**变量描述**（如 `抓 authorization 里的值`，`q` 退出）
5. 选择**分隔符**（输入序号，`q` 退出）：
   1. `newline` — 换行拼接为同一个变量
   2. `single` — 每个账号存为独立变量
   3. `#` — 用 # 拼接
   4. `@` — 用 @ 拼接
   5. `&` — 用 & 拼接

### `/查询`
包含两种查询方式：
1. **查询项目** — 选择项目查看其下所有账号及变量值（支持 `0` 查看全部）
2. **查询账号** — 选择账号备注名查看其出现在哪些项目中（支持 `0` 查看全部）

### `/管理项目`
包含两种删除方式：
1. **删除项目中的账号** — 选择项目 → 选择要删除的账号（支持 `0` 全选）
2. **删除账号中的项目** — 选择账号备注名 → 从所有包含该账号的项目中删除（支持 `0` 全选）

两种操作都会自动同步青龙面板。

### `/删除项目`
删除整个项目及其所有账号数据（支持 `0` 全选）。

### `/项目帮助`
查看所有命令的详细用法。输入序号查看对应命令说明。

## 主菜单 vs 直接指令

所有功能可以通过 `/项目` 主菜单进入，也可以通过直接指令调用：

| 功能 | 直接指令 |
|------|----------|
| 项目列表 | `/项目列表` |
| 新增项目 | `/新增项目` |
| 查询 | `/查询` |
| 管理项目 | `/管理项目` |
| 删除项目 | `/删除项目` |
| 项目帮助 | `/项目帮助` |

## 数据存储

插件数据保存在 `plugin_data/` 目录下：

- `projects.json` — 项目、账号数据
- `config.json` — 插件配置（保留字段）

## 注意事项

- 同步操作会删除青龙面板中同名的环境变量后重新创建
- 所有交互输入 `q` 退出，输入 `0` 全选（仅删除/查询命令支持）
- 每次输入等待 120 秒，超时自动退出
- 项目列表按青龙实例分组显示，选项目时直接输入全局序号：
  ```
  【青龙面板1】
   1.京东
   2.淘宝
  【青龙面板2】
   3.百度
   4.腾讯
  ```

## 更新记录

### v1.0.0
- 初始版本：项目/账号管理，青龙同步

### 优化记录
- **重构 session 模式**：一个命令注册一个 session_waiter，用 `ctl.keep()` 维持多轮对话，替代多层嵌套
- **超时延长**：等待时间从 60s 改为 120s
- **多实例支持**：`_conf_schema.json` 使用 `template_list` 类型，WebUI 支持添加多个青龙实例
- **防御性编程**：配置解析增加 `isinstance` 类型检查，兼容新旧配置格式
- **HTTP 客户端优化**：`QinglongClient` 使用共享连接、超时重试、token 自动刷新
- **`/项目` 主菜单**：统一入口，6 个选项涵盖所有功能
- **项目列表分组显示**：项目按青龙实例分组展示
- **`/查询` 命令**：合并查询项目和查询账号
- **`/管理项目` 扩展**：新增「删除账号中的项目」，支持跨项目批量删除
- **`/删除项目` 支持全选**：输入 0 删除全部项目
- **退出键改为 `q`**：所有命令的退出由 `0` 改为 `q`
- **`0` 全选**：删除和查询命令支持输入 `0` 选择全部
- **分隔符改为序号选择**：新增项目时分隔符改为 1-5 序号选择
- **变量描述**：新增项目支持输入变量描述，添加账号时自动显示提示
